# Python Fantasy - Knight's Journey Skills

This document outlines the skills and knowledge paths for the Python Fantasy educational application.

## Learning Path (Quests)

| Quest | Title | Description | Item Reward | Key Concepts |
| :--- | :--- | :--- | :--- | :--- |
| 01 | ปฐมบทแห่งอัศวิน | แนะนำรายวิชา | เข็มทิศทองคำ 🧭 | Course Introduction, Basic Programming Logic |
| 02 | แผนที่ยุทธศาสตร์ | Flowchart | แผนที่ยุทธวิธี 🗺️ | Algorithm Planning, Flowchart Symbols |
| 03 | เสียงประกาศของนักรบ | Print | แตรศึก 📢 | `print()`, Strings, Escaping, `sep`, `end` |
| 04 | คลังอาวุธและเสบียง | Variables | ถุงเป้เดินทาง 🎒 | Variable Assignment, Naming Conventions, Basic Data Types |
| 05 | ศาสตร์การคำนวณพลัง | Operator | ลูกคิดยุทธการ 🧮 | Arithmetic Operators, Modulo, Power (`**`), Order of Operations |
| 06 | ทางเลือกแห่งเกียรติยศ | Condition | โล่สะท้อนสัจจะ 🛡️ | `if`, `elif`, `else`, Boolean Logic |
| 07 | การฝึกฝนที่ไร้สิ้นสุด | Loop | นาฬิกาทรายนิรันดร์ ⏳ | `for` loops, `while` loops, `range()` |
| 08 | กระบวนท่าไม้ตาย | Function | คัมภีร์ลับ 📜 | `def`, Function Call, Parameters, `return` |
| 09 | หีบสมบัติใต้ปราสาท | Collections | หีบเก็บไอเทม 📦 | Lists `[]`, Tuples `()`, Dictionaries `{}` |
| 10 | การสร้างอาณาจักรมายา | GUI Development | ลูกแก้ววิเศษ 🔮 | Graphical User Interface, `tkinter` Basics |

## Technical Stack
- **Frontend:** HTML, Tailwind CSS, Font Awesome
- **Python Runtime:** Pyodide (Client-side Python)
- **Backend/Database:** Firebase (Firestore & Authentication)
- **AI Integration:** Typhoon API (via OpenAI-compatible SDK)
